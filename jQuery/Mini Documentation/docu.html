My Mini Documentation

Click

Attaches an event handler that runs a function whenever the selected element is clicked. Commonly used to trigger actions like showing, hiding, or modifying elements.

Hide

Hides the selected elements by setting their CSS display property to none. The element stays in the DOM but becomes invisible.

Show

Displays hidden elements by removing the display: none property or applying the default display for that element.

Toggle

Switches the visibility of elements; it hides visible elements and shows hidden ones with a single function.

Slide Down

Reveals an element with a smooth sliding animation from top to bottom. Often used for dropdowns or collapsible sections.

Slide Up

Hides an element with a sliding-up animation, gradually collapsing the element upwards until it disappears.

Slide Toggle

Combines slideDown() and slideUp(). It toggles the sliding animation based on whether the element is currently visible or hidden.

Fade In

Gradually changes the opacity of an element from hidden to visible, creating a smooth fade-in effect.

Fade Out

Gradually changes the opacity from visible to hidden, making the element fade out smoothly.

Add Class

Adds one or more CSS classes to the selected elements. Useful for dynamically applying styles or triggering CSS-based animations.

Before

Inserts HTML, text, or elements before the selected element in the DOM tree.

After

Inserts HTML, text, or elements after the selected element in the DOM tree.

Append

Inserts content inside the selected element as its last child. Commonly used to dynamically add elements or content.

HTML

Gets the inner HTML content of the first selected element or sets new HTML content for all matched elements.

Attr

Gets the value of an attribute from the first matched element, or sets one or more attributes for all selected elements.

Value

Gets the current value of form elements (like input fields, selects, and textareas) or sets a new value.

Text

Gets the combined text content of the selected elements or sets new text content, ignoring any HTML tags.
