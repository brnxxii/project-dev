Here we would like to show the very basics of PHP in a short, simple tutorial. This text only deals iwht dynamic web page creation with PHP, though PHP is not only capable of creating web pages. See the section titled What can PHP do? for more information.

PHP is a server-side scripting language, which means that the PHP code is executed on the server and the result is sent to the client. The client only sees the result of the PHP code, not the code itself.
